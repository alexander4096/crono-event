Crono-Event
Version 1.0.0

This program is a timer for events and activities that allows you to set the time and set alarms.

What the program allows you to do:

- Track time during streaming activities
- Manage time for topics to be covered between classes/presentations/conferences
- Assist the user with notifications of activities that need to be completed
- Indicate the completion of an action with alarms.

Program Content:
- Time
- Countdown timer
- Countdown timer
- Event timer for activities
- Break history
- Gradual time alarm
